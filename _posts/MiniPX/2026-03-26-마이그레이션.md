## 이미지 마이그레이션

*   **TO-BE server**

```bash
# 디렉토리 생성
sudo mkdir -p /opt/minipx/img
sudo chown -R ec2-user:ec2-user /opt/minipx
```



* **Local(Power Shell)**

``` bash
# 오늘 날짜를 YYYY-MM-DD 형식으로 구하기
$today = (Get-Date).ToString("yyyy-MM-dd")

# 로컬 경로 생성
$local_path = "C:\Users\KimSangMin\git\miniPX\ignore\$today"

# 해당 경로가 없으면 디렉토리 생성
if (-not (Test-Path $local_path)) {
    New-Item -ItemType Directory -Path $local_path
}

# SCP 명령어 실행 (AS-IS서버에서 로컬로 파일 복사)
scp -i "C:\Users\KimSangMin\git\miniPX\mpxLive2026.pem" -r ec2-user@43.200.173.163:/opt/minipx $local_path

# SCP 명령어 실행 (로컬에서 TO-BE서버로 파일 복사)
scp -i "C:\Users\KimSangMin\git\miniPX\ignore\mpx2026_3.pem" -r "$local_path/minipx/img/*" ec2-user@43.202.50.109:/opt/minipx/img
```
