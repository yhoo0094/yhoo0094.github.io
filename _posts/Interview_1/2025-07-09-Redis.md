## Redis(Remote Dictionary Server)

| 항목                   | 설명                                                         |
| ---------------------- | ------------------------------------------------------------ |
| 🧠 **In-Memory**        | 모든 데이터를 RAM에 저장하여 매우 빠른 처리 속도를 자랑      |
| ⚙️ **Key-Value 구조**   | 키 하나에 다양한 타입의 값을 저장 (문자열, 리스트, 해시, 집합 등) |
| 💾 **Persistence**      | 메모리 기반이지만 디스크에 백업 가능 (RDB, AOF 방식)         |
| 🔁 **Replication & HA** | 마스터-슬레이브 복제, Sentinel, Cluster 구성 지원            |
| 📣 **Pub/Sub**          | 채널을 통한 메시징 시스템 기능 지원                          |
| 🧪 **Atomic 연산 지원** | 멀티스레드 없이도 안전하게 연산 처리                         |
